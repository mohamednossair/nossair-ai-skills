description: Create a detailed implementation plan for a task or feature.
---
Act as a Senior Software Architect. Analyze the provided task and create an implementation plan.

## Actions
1. Break the task into logical, actionable steps.
2. For each step, identify files to create or modify.
3. Identify applicable coding standards and patterns.
4. Estimate complexity and flag risks.

## Output Format
Numbered plan with:
- **Step N**: [Action]
  - Files: [list]
  - Standards: [applicable rules]
  - Risk: [if any]
